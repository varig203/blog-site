Section 1: Setting up the environment

Update Void Linux!

Before doing anything, update your system. Assuming you're using the default setup (with sudo), run:


sudo xbps-install -Su

If you use doas instead of sudo, just swap the command.

Add the Hyprland Repository

Choose what fits your system best.

Options What it is
x86_64 x86 64 bit glibc
x86_64-musl x86 64 bit Musl
aarch64 ARM 64 bit glibc
aarch64-musl ARM 64 bit Musl

If you are not sure just run:


xbps-uhelper arch
ldd --version

Just align yourself with the table with the following step.


Run this following command be sure to replace the architecture string at the end if you're not using x86_64-glibc. Small disclaimer that this is a third party repository.


echo repository=https://raw.githubusercontent.com/Encoded14/void-extra/repository-x86_64 | sudo tee /etc/xbps.d/20-repository-extra.conf

Section 2: Install Hyprland

Installing

Check if Hyprland appears in your repo:


xbps-query -Rs hypr

If it does, you're good to go! Install Hyprland and its xdg portal. If not double check if you ran it correctly.


sudo xbps-install -S hyprland xdg-desktop-portal-hyprland

NOTE: If you use a Nvidia GPU, refer to the Hyprland Wiki's Nvidia section for extra setup steps.
